### preview-img-zwj

原生js实现的图片放大组件

### 支持功能

1. 图片缩放
2. 图片旋转
3. 图片移动

## 使用

### use npm

```
npm i preview-img-zwj -d

```

```
import previewImg from 'preview-img-zwj'
// Add on click event
previewImg({src:url})

```

    npm run publish
